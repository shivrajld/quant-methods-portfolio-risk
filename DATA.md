# Data and Model Notes

The Excel workbook is the primary model artifact for this project.

## Workbook

`model/quant_methods_portfolio_model.xlsx` contains:

- Raw monthly asset and factor return data
- Asset summary statistics and distribution checks
- Factor regressions and rolling beta analysis
- Variance inflation diagnostics for factor selection
- Historical wealth simulations from 2004 to 2013
- Mean variance optimization with allocation and risk contribution constraints
- Risk parity analysis using inverse volatility weights
- Total risk contribution sheets for proposed portfolios
- Final simulation pages for MVO and risk parity portfolios

The copied workbook metadata has been normalized for public release. The workbook formulas, sheet names, and outputs were otherwise preserved.

## Presentation

The original presentation was used as reference material for the README and result summary. It is not included because it contains private template language and personal contact information.

## Figures

The `figures` folder contains exported charts from the workbook and presentation workflow. These are included so the GitHub page shows the core risk story without requiring someone to open Excel first.
